# web — Homepage, UCP & ACP (Phase 5)

Dieses Verzeichnis nimmt in Phase 5 die Next.js-Anwendung auf
(Homepage, User Control Panel, Admin Control Panel).

Bewusst noch leer: Phase 1 liefert zuerst das Fundament (Log-Pipeline,
Framework-Core, Datenmodell), damit UCP/ACP später auf einer vollständigen

Session-Replay, deren Daten bereits ab Phase 1 lückenlos erfasst werden.

Bis dahin liefert der Reverse Proxy (`proxy/Caddyfile`) einen Platzhalter aus.
